# SR_Project_H414
### Requirements
Requirements are all on this web page : https://github.com/ilpincy/argos3
### Compilation
$ git clone https://github.com/ilpincy/argos3.git argos3  
$ cd argos3  
$ mkdir build_simulator  
$ cd build_simulator  
$ cmake ../src  
$ make  
### Line to execute the code
argos3 -c foraging.argos
